⚡ Recursos Principais
Tecnologias e capacidades do serviço de validação
🔍

OCR Multi-Estratégia

10 estratégias de OCR com Tesseract: rotação 0°, 90°, 180°, 270°, escala 2× e contraste adaptativo. Seleciona automaticamente o melhor resultado.

🪪

Extração MRZ (CNH-e)

Lê o código de barras MRZ da CNH digital para extrair nome completo e data de validade com precisão, mesmo quando o OCR falha no texto impresso.

🏫

Lista Oficial BRF

Verifica se o certificado de Direção Defensiva foi emitido por escola homologada pela BRF. Regra diferenciada: SEST SENAT mínimo 4h, demais mínimo 8h.

🩺

Validação ASO

Extrai nome, empresa, resultado (Apto/Inapto), data de realização e calcula validade automática (6, 12 ou 24 meses) conforme marcação no documento.

📅

Verificação de Vencimento

Compara datas de validade com a data atual. CNH vencida, ASO expirado e certificado de direção defensiva com mais de 5 anos são automaticamente reprovados.

🔗

API REST + Base64

Endpoint único via POST com arquivo em Base64. Suporta PDF (convertido automaticamente) e imagens JPG/PNG. Resposta JSON estruturada com todos os dados extraídos.

import base64, requests def validar_documento(caminho_arquivo, tipo): with open(caminho_arquivo, "rb") as f: b64 = base64.b64encode(f.read()).decode() r = requests.post( "http://seu-servidor:5000/api/validation", json={ "tipoDocumento": tipo, "base64Arquivo": b64 }, timeout=60 ) return r.json() # Exemplos de uso resultado = validar_documento("CNH.pdf", "CNH") resultado = validar_documento("ASO.jpg", "ASO") resultado = validar_documento("cert.pdf", "DirecaoDefensiva") print(resultado["status"]) # APROVADO | REPROVADO | ANÁLISE MANUAL print(resultado["confianca"]) # 0–100
